Basic Python

Syntax, variables, control flow and example programs
Difficulty: Beginner

1. Hello & Variables

Python uses simple syntax and indentation. Save code in a file `hello.py` and run with python hello.py.

Example

# hello.py
name = input("What's your name? ")
print("Hello,", name)

# basic types
age = 25               # integer
pi = 3.14159           # float
is_student = True      # boolean

2. Control Flow

Use if, for and while for logic and loops.

Example — sum of numbers

# sum_n.py
n = int(input("n = "))
total = 0
for i in range(1, n+1):
    total += i
print("Sum is", total)

Exercise

Write a program that reads a list of numbers (space separated) and prints the maximum and minimum.

Hints
  • Use split() and map(int, ...).
  • Use built-in max() and min().

Basic Java

Classes, main method, types and simple I/O
Difficulty: Beginner

1. Hello World & Structure

Java code is organized in classes. Compile with javac then run with java.

Example

// Hello.java
import java.util.Scanner;
public class Hello {
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        System.out.print("Name: ");
        String name = sc.nextLine();
        System.out.println("Hello, " + name);
    }
}

2. Loops & Arrays

Example — sum array

int[] arr = {1,2,3,4,5};
int sum = 0;
for (int v : arr) sum += v;
System.out.println("Sum = " + sum);

SQL Basics

CREATE, SELECT, WHERE, JOIN & aggregation
Difficulty: Beginner

1. SELECT & WHERE

Query rows from tables. Example below uses a table students(id, name, score).

Examples

-- select all
SELECT * FROM students;

-- filter
SELECT name, score FROM students WHERE score >= 60 ORDER BY score DESC;

-- top 3
SELECT name FROM students ORDER BY score DESC LIMIT 3;

2. JOIN & Aggregation

Example — join

-- assume courses(course_id, title) and enroll(student_id, course_id)
SELECT s.name, c.title
FROM students s
JOIN enroll e ON s.id = e.student_id
JOIN courses c ON e.course_id = c.course_id;

-- average score
SELECT AVG(score) as avg_score FROM students;

DSA — Introduction

Complexity, arrays, linked lists, stacks, queues, sorting basics
Difficulty: Beginner → Intermediate

1. Time Complexity

Common complexity classes:

  • O(1) — constant
  • O(n) — linear
  • O(n log n) — typical sorting
  • O(n²) — nested loops

2. Example — Binary Search (Python)

Binary Search

def binary_search(arr, target):
    lo, hi = 0, len(arr)-1
    while lo <= hi:
        mid = (lo + hi) // 2
        if arr[mid] == target:
            return mid
        if arr[mid] < target:
            lo = mid + 1
        else:
            hi = mid - 1
    return -1

Complexity: O(log n)
